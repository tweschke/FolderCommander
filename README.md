# Folder Commander

A macOS app for creating and managing folder-structure templates and generating projects from them.

## Overview

Folder Commander lets you:

- **Create templates** — Define folder structures in a visual or text editor.
- **Create projects** — Run a wizard to pick a template, name the project, choose a location, and generate the folder structure.
- **Back up templates** — Import and export all templates as a single JSON file (Backups in the sidebar).

Built with SwiftUI for macOS 14.0+.

## Build and run

1. Open `Folder Commander.xcodeproj` in Xcode.
2. Select the **Folder Commander** scheme and a Mac destination.
3. Build and run (⌘R).
